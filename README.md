---------- PROJET YDAYS : FACTEURS DE RISQUE POUR ASTEROÏDES ----------

L'objecti est de réaliser une interface interactive (type PowerBI) pour évaluer les facteurs qui rendent un objet astronomique potentiellement dangereux ou non.
On finira le projet par une conclusion en se servant des données explorées et des connaissances personnelles pour établir les facteurs de risque.
